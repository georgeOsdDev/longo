http://docs.mongodb.org/manual/reference/method/db.collection.find/#definition

The projection parameter takes a document of the following form: { field1: , field2: ... } The value can be any of the following: - 1 or true to include the field. The find() method always includes the _id field even if the field is not explicitly stated to return in the projection parameter.

  • 0 or false to exclude the field. A projection cannot contain both include and exclude specifications, except for the exclusion of the _id field. In projections that explicitly include fields, the _id field is the only field that you can explicitly exclude.

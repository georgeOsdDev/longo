GettingStarted

Usage

Get Longo Components

  1. Install node.js
  2. Required grunt the javascript task runner
    • Install grunt: npm install -g grunt-cli
    • Install bower: npm install -g bower
  3. Clone latest soruce code from GitHub
  4. Build code cd longo && npm install && grunt build

Import Your Application

Make sure that longo.js, longoWorker.js, lib is put in same directory. Default root name is Longo.js.

This tree will be created by grunt compile task as dest directory.

Longo.js
├── lib
│   ├── underscore
│   │   ├── underscore-min.js
│   │   ├ ...
│   │
│   └── underscore-query
│       ├── lib
│       │   ├── underscore-query.min.js
│       │   ├ ...
│       ├ ...
│
├── longo.js
├── longo.min.js
├── longo.min.map
├── longoWorker.js
├── longoWorker.min.js
└── longoWorker.min.map

longo.js (or longo.min.js) underscore.js are need to be loaded in html.

Other files will be imported at worker thread.

%lt;script type="text/javascript" src="path/to/Longo.js/lib/underscore-min.js">%lt;/script>
%lt;script type="text/javascript" src="path/to/Longo.js/longo.js">%lt;/script>
%lt;script>
    // Adjust root directory.
    Longo.setRoot("path/to/Longo.js");
%lt;script>
